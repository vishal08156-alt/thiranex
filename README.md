# 📊 Data Cleaning & Visualization Project

## 📌 Project Overview
This project focuses on cleaning, preprocessing, and visualizing a raw dataset using Python. The goal is to transform raw data into meaningful insights through data cleaning techniques and exploratory data analysis (EDA).

---

## 🎯 Objective
- Clean and preprocess the dataset.
- Handle missing values, duplicates, and invalid data.
- Convert columns to appropriate data types.
- Perform exploratory data analysis (EDA).
- Visualize key insights using charts and graphs.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset
The project uses the **Superstore Orders Dataset**, which contains information about:
- Orders
- Sales
- Profit
- Quantity
- Discount
- Shipping Cost
- Categories
- Order & Shipping Dates

---

## 🔍 Data Cleaning Steps
- Converted numeric columns to proper data types.
- Converted date columns into datetime format.
- Checked dataset information and summary statistics.
- Identified and removed duplicate records.
- Handled missing values.
- Prepared the dataset for analysis.

---

## 📈 Visualizations Created
- Box Plot for Outlier Detection
- Histogram of Numeric Features
- Correlation Heatmap
- Category-wise Order Count
- Sales vs Profit Scatter Plot

---

## 📊 Key Insights
- Cleaned and preprocessed the dataset successfully.
- Removed duplicate records and handled missing values.
- Explored relationships between sales, profit, and discounts.
- Identified data distribution using histograms.
- Compared order counts across different product categories.
- Visualized the correlation among numerical features.
